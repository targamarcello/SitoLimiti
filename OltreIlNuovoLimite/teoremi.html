1. Teoremi sulle operazioni con i limiti

I seguenti teoremi ci permettono di calcolare i limiti di espressioni complesse scomponendole in operazioni più semplici.

Teorema della somma

Se esistono limx→x0 f(x) = L e limx→x0 g(x) = M, allora:

limx→x0 [f(x) + g(x)] = limx→x0 f(x) + limx→x0 g(x) = L + M

Questo teorema si estende anche alla sottrazione.

Teorema del prodotto

Se esistono limx→x0 f(x) = L e limx→x0 g(x) = M, allora:

limx→x0 [f(x) · g(x)] = limx→x0 f(x) · limx→x0 g(x) = L · M

Teorema del quoziente

Se esistono limx→x0 f(x) = L e limx→x0 g(x) = M con M ≠ 0, allora:

limx→x0 [f(x) / g(x)] = limx→x0 f(x) / limx→x0 g(x) = L / M

Se M = 0 e L ≠ 0, il limite non esiste o è infinito (dipende dal segno di f(x) e g(x)).

Teorema della potenza

Se esiste limx→x0 f(x) = L e n è un numero intero positivo, allora:

limx→x0 [f(x)]n = [limx→x0 f(x)]n = Ln

Se n è un numero intero negativo e L ≠ 0, allora:

limx→x0 [f(x)]n = [limx→x0 f(x)]n = Ln = 1 / L|n|

Esempio

Calcolare limx→2 (3x² - x + 4) / (x - 1)

Soluzione:

limx→2 (3x² - x + 4) / (x - 1) = [limx→2 (3x² - x + 4)] / [limx→2 (x - 1)]

= [3·limx→2 x² - limx→2 x + 4] / [limx→2 x - 1]

= [3·2² - 2 + 4] / [2 - 1]

= [12 - 2 + 4] / 1

= 14

2. Teoremi del confronto

Questi teoremi ci permettono di determinare il valore di un limite confrontandolo con altri limiti già noti.

Teorema dei due carabinieri (o del confronto)

Se in un intorno di c (escluso eventualmente c) si ha:

g(x) ≤ f(x) ≤ h(x)

e se limx→x0 g(x) = limx→x0 h(x) = L, allora anche limx→x0 f(x) = L.

Teorema del confronto per limiti infiniti

Se in un intorno di c (escluso eventualmente c) si ha f(x) ≥ g(x) e limx→x0 g(x) = +∞, allora anche limx→x0 f(x) = +∞.

Analogamente, se f(x) ≤ g(x) e limx→x0 g(x) = -∞, allora anche limx→x0 f(x) = -∞.

Esempio

Consideriamo le funzioni g(x) = x², f(x) = x² + sin(x)/x e h(x) = x² + 1/x per x che tende a +∞.

Sappiamo che sin(x)/x è limitata da -1/x a 1/x per x sufficientemente grande, quindi:

x² - 1/x ≤ x² + sin(x)/x ≤ x² + 1/x

Ora, limx→+∞ (x² - 1/x) = limx→+∞ (x² + 1/x) = +∞

Per il teorema del confronto, possiamo concludere che limx→+∞ (x² + sin(x)/x) = +∞

3. Teorema della permanenza del segno

Questo teorema ci permette di stabilire il segno di una funzione in prossimità di un punto conoscendo il suo limite.

Teorema della permanenza del segno

Se limx→x0 f(x) = L e L > 0, allora esiste un intorno di c in cui f(x) > 0 per ogni x dell'intorno (diverso eventualmente da c).

Analogamente, se L < 0, allora esiste un intorno di c in cui f(x) < 0 per ogni x dell'intorno (diverso eventualmente da c).

Nota importante

Il teorema della permanenza del segno è utile nelle dimostrazioni che riguardano le funzioni continue e nell'analisi degli intervalli in cui una funzione mantiene un segno costante.

4. Teorema di unicità del limite

Questo teorema garantisce che, se esiste, il limite di una funzione in un punto è unico.

Teorema di unicità del limite

Se limx→x0 f(x) = L e limx→x0 f(x) = M, allora L = M.

In altre parole, una funzione non può avere due limiti diversi nello stesso punto.

Dimostrazione per assurdo

Si può dimostrare questo teorema per assurdo. Supponiamo che L ≠ M e poniamo ε = |L - M|/2 > 0.

Per la definizione di limite, esistono δ₁ > 0 e δ₂ > 0 tali che:

|f(x) - L| < ε per 0 < |x - c| < δ₁ e |f(x) - M| < ε per 0 < |x - c| < δ₂

Prendendo δ = min{δ₁, δ₂} e x tale che 0 < |x - c| < δ, si ha contemporaneamente:

|f(x) - L| < ε e |f(x) - M| < ε

Ma allora |L - M| = |L - f(x) + f(x) - M| ≤ |L - f(x)| + |f(x) - M| < 2ε = |L - M|

Questa è una contraddizione, quindi deve essere L = M.

5. Teoremi di esistenza

Questi teoremi ci permettono di stabilire l'esistenza di un limite senza dover necessariamente calcolarne il valore.

Teorema di esistenza del limite per funzioni monotone

Se f(x) è monotona in un intorno destro di c e limitata superiormente, allora esiste limx→x0⁺ f(x).

Analogamente, se f(x) è monotona in un intorno sinistro di c e limitata inferiormente, allora esiste limx→x0⁻ f(x).

Teorema di esistenza del limite per funzioni composte

Se esiste limx→x0 g(x) = L e se la funzione f è continua in L, allora:

limx→x0 f(g(x)) = f(limx→x0 g(x)) = f(L)

Esempio

Calcolare limx→0 sin(3x²)

Soluzione:

Possiamo applicare il teorema delle funzioni composte con g(x) = 3x² e f(y) = sin(y).

Sappiamo che limx→0 3x² = 0 e che sin(y) è continua in y = 0, quindi:

limx→0 sin(3x²) = sin(limx→0 3x²) = sin(0) = 0

6. Limiti notevoli

I limiti notevoli sono limiti di funzioni particolari che ricorrono frequentemente nei calcoli e che è utile memorizzare.

Primo limite notevole

limx→0 sin(x) / x = 1

Questo limite è fondamentale nella trigonometria e nell'analisi matematica.

Secondo limite notevole

limx→0 (1 - cos(x)) / x² = 1/2

Questo limite è utile per lo studio di funzioni che coinvolgono il coseno.

Limiti esponenziali notevoli

limx→∞ (1 + 1/x)x = e
limx→0 (1 + x)1/x = e

Questi limiti sono alla base della definizione del numero di Eulero e.

Altri limiti notevoli

limx→0 ln(1 + x) / x = 1
limx→0 (ex - 1) / x = 1
limx→0 tan(x) / x = 1
limx→0 arcsin(x) / x = 1
limx→0 arctan(x) / x = 1

Esempio

Calcolare limx→0 sin(5x) / (3x)

Soluzione:

limx→0 sin(5x) / (3x) = limx→0 [sin(5x) / (5x) · (5x) / (3x)] = limx→0 [sin(5x) / (5x)] · limx→0 [(5x) / (3x)]

Utilizzando il primo limite notevole e sostituendo y = 5x (se x → 0, anche y → 0):

limx→0 [sin(5x) / (5x)] · limx→0 [(5x) / (3x)] = 1 · (5/3) = 5/3
