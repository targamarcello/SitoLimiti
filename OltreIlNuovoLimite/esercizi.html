Esercizio 1: Limite di una funzione razionale Facile

Calcola il seguente limite:

limx→3 (2x + 5)

Soluzione

In questo caso, possiamo applicare direttamente la definizione di limite, sostituendo il valore di x a cui tende (3) nella funzione.

limx→3 (2x + 5) = 2(3) + 5

= 6 + 5

= 11

Quindi, limx→3 (2x + 5) = 11

Esercizio 2: Limite con forma indeterminata 0/0 Medio

Calcola il seguente limite:

limx→2 (x² - 4) / (x - 2)

Soluzione

Questo limite presenta una forma indeterminata del tipo 0/0. Verificiamo:

limx→2 (x² - 4) / (x - 2)

Per x = 2, otteniamo (2² - 4) / (2 - 2) = 0/0, che è una forma indeterminata.

Possiamo risolvere questo limite fattorizzando il numeratore:

(x² - 4) = (x - 2)(x + 2)

Sostituendo nella formula del limite:

limx→2 (x² - 4) / (x - 2) = limx→2 [(x - 2)(x + 2)] / (x - 2)

Semplificando il fattore comune (x - 2):

limx→2 (x + 2) = 2 + 2 = 4

Quindi, limx→2 (x² - 4) / (x - 2) = 4

Esercizio 3: Limite all'infinito Difficile

Calcola il seguente limite:

limx→∞ (3x² + 2x - 1) / (x² - 5x + 7)

Soluzione

Quando abbiamo un limite all'infinito di funzioni razionali, possiamo dividerle per la più alta potenza di x presente al denominatore:

limx→∞ (3x² + 2x - 1) / (x² - 5x + 7)

Dividiamo numeratore e denominatore per x²:

limx→∞ [(3x²/x²) + (2x/x²) - (1/x²)] / [(x²/x²) - (5x/x²) + (7/x²)]

= limx→∞ [3 + (2/x) - (1/x²)] / [1 - (5/x) + (7/x²)]

Quando x tende all'infinito, 1/x e 1/x² tendono a 0, quindi:

limx→∞ [3 + 0 - 0] / [1 - 0 + 0] = 3/1 = 3

Quindi, limx→∞ (3x² + 2x - 1) / (x² - 5x + 7) = 3

Esercizio 4: Limite con forma indeterminata ∞/∞ Medio

Calcola il seguente limite:

limx→∞ (2x² + 3x) / (4x + 1)

Soluzione

Questo limite presenta una forma indeterminata del tipo ∞/∞.

Per risolvere questo tipo di limiti, possiamo dividere sia il numeratore che il denominatore per la più alta potenza di x presente in uno dei due.

In questo caso, la più alta potenza è x² al numeratore. Dividiamo tutto per x:

limx→∞ (2x² + 3x) / (4x + 1) = limx→∞ [(2x² / x) + (3x / x)] / [(4x / x) + (1 / x)]

= limx→∞ (2x + 3) / (4 + 1/x)

Quando x tende all'infinito, 1/x tende a 0, quindi:

limx→∞ (2x + 3) / (4 + 0)

L'espressione diventa (2x + 3) / 4

Quando x tende all'infinito, 2x diventa infinitamente più grande di 3, quindi:

limx→∞ (2x) / 4 = limx→∞ x/2 = ∞

Quindi, limx→∞ (2x² + 3x) / (4x + 1) = ∞

Esercizio 5: Limite elementare Facile

Calcola il seguente limite:

limx→1 (x³ - 3x + 5)

Soluzione

Possiamo calcolare direttamente questo limite sostituendo x = 1 nella funzione:

limx→1 (x³ - 3x + 5) = 1³ - 3(1) + 5

= 1 - 3 + 5

= 3

Quindi, limx→1 (x³ - 3x + 5) = 3
