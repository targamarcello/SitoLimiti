Definizione di Limite

Comprendere il concetto fondamentale dell'analisi matematica

Cos'è un limite?

Il limite di una funzione è un concetto matematico che descrive il comportamento della funzione quando la sua variabile indipendente si avvicina a un determinato valore. In termini più formali, il limite di una funzione f(x) quando x tende a un valore c è il valore a cui f(x) si avvicina, se esiste.

limx→x0 f(x) = L

Questa notazione indica che quando x si avvicina a c (ma non è necessariamente uguale a c), il valore della funzione f(x) si avvicina al valore L.

Definizione formale (ε-δ)

La definizione formale di limite utilizza il concetto di vicinanza tramite i valori ε (epsilon) e δ (delta):

limx→x0 f(x) = L significa che ∀ ε > 0, ∃ δ > 0 tale che
se 0 < |x - x0| < δ, allora |f(x) - L| < ε

In parole semplici, questa definizione afferma che possiamo rendere f(x) arbitrariamente vicino a L (con un margine di errore ε) scegliendo x sufficientemente vicino a c (entro una distanza δ).

Esempio di applicazione della definizione ε-δ

Verifichiamo che limx→2 (3x - 1) = 5 usando la definizione ε-δ.

Vogliamo trovare δ in funzione di ε tale che se |x - 2| < δ, allora |(3x - 1) - 5| < ε.

Semplifichiamo: |(3x - 1) - 5| = |3x - 6| = 3|x - 2|

Quindi, se scegliamo δ = ε/3, avremo che se |x - 2| < δ = ε/3, allora |(3x - 1) - 5| = 3|x - 2| < 3(ε/3) = ε

Questo dimostra che il limite è effettivamente 5.

Tipi di limiti

Limite destro e sinistro

Il limite destro considera i valori di x che si avvicinano a c da destra (x > c), mentre il limite sinistro considera i valori di x che si avvicinano a c da sinistra (x < c).

limx→x0+ f(x) = limite destro
limx→x0- f(x) = limite sinistro

Un limite esiste solo se i limiti destro e sinistro sono uguali.

Limite all'infinito

Studia il comportamento di una funzione quando x tende a infinito (positivo o negativo).

limx→∞ f(x) = L
limx→-∞ f(x) = M

Questi limiti sono utili per studiare il comportamento asintotico delle funzioni.

Limite infinito

Si verifica quando il valore della funzione cresce indefinitamente in valore assoluto mentre x si avvicina a c.

limx→x0 f(x) = ∞
limx→x0 f(x) = -∞

Questi limiti indicano la presenza di asintoti verticali nel grafico della funzione.

Proprietà dei limiti

I limiti seguono diverse proprietà algebriche che ne facilitano il calcolo:

  • Limite di una somma: lim[f(x) + g(x)] = lim f(x) + lim g(x)
  • Limite di un prodotto: lim[f(x) · g(x)] = lim f(x) · lim g(x)
  • Limite di un quoziente: lim[f(x)/g(x)] = lim f(x) / lim g(x), se lim g(x) ≠ 0
  • Limite di una potenza: lim[f(x)]n = [lim f(x)]n
  • Limite di una radice: lim √f(x) = √lim f(x), se lim f(x) ≥ 0

Queste proprietà sono discusse più approfonditamente nella sezione dei teoremi.
