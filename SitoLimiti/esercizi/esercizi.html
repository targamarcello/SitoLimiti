Calcola il Limite - Esempi pratici

Esercizio 1: Limite di (x² - 1) / (x - 1) quando x tende a 1

Per calcolare questo limite, possiamo semplificare l'espressione come segue:

(x² - 1) / (x - 1) = (x - 1)(x + 1) / (x - 1)

Eliminando il fattore comune (x - 1), otteniamo:

x + 1

Quando x tende a 1, il limite sarà 2.

Esercizio 2: Limite di sin(x) / x quando x tende a 0

Questo è un limite fondamentale che tende a 1. La risposta è nota:

lim(x → 0) sin(x) / x = 1

Questo limite è fondamentale nelle serie di Taylor e nelle funzioni trigonometriche.

Esercizio 3: Limite di (e^x - 1) / x quando x tende a 0

Questo è un esempio di limite fondamentale che dà come risultato 1:

lim(x → 0) (e^x - 1) / x = 1

Questo limite è usato frequentemente nelle applicazioni esponenziali e nelle serie di Taylor.
