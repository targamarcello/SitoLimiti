Esercizi Pratici sui Limiti

Per aiutarti a capire meglio come calcolare i limiti, proponiamo una serie di esercizi pratici divisi per difficoltà:

Esercizi Base

  • 1. Calcolare il limite di: f(x) = (x² - 4) / (x - 2) per x → 2

    • Indica se puoi semplificare la funzione e applicare direttamente il limite o se è necessaria un'altra tecnica.

  • 2. Determinare il limite di: g(x) = e^x / x per x → ∞

    • Usa il comportamento asintotico per determinare il limite all'infinito.

  • 3. Calcolare il limite di: h(x) = sin(x) / x per x → 0

    • Un classico limite noto che può essere calcolato usando tecniche di analisi.

Esercizi Intermedi

  • 4. Calcolare il limite di: f(x) = (x³ - 8) / (x - 2) per x → 2

    • Applica il teorema di L'Hopital se necessario o prova a fattorizzare.

  • 5. Calcolare il limite di: g(x) = (x² - 1) / (x - 1) per x → 1

    • Verifica se il limite si può calcolare con la sostituzione diretta o se c'è bisogno di semplificare.

Esercizi Avanzati

  • 6. Calcolare il limite di: f(x) = (x^3 + 3x² - x - 3) / (x² - 1) per x → 1

    • Questo esercizio richiede l'uso di tecniche avanzate come la divisione polinomiale o il teorema di L'Hopital.

  • 7. Calcolare il limite di: g(x) = (x² - 2x + 1) / (x - 1) per x → 1

    • Questo esercizio esplora i limiti con forme indeterminate e richiede una buona comprensione delle tecniche di calcolo.
