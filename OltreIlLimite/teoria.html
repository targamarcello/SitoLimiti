Teoria dei Limiti

Il limite è un concetto fondamentale nell'analisi matematica che descrive il comportamento di una funzione quando la variabile indipendente si avvicina a un certo valore. In particolare, X0 è FINITO ed L è INFINITO rappresenta il caso in cui una funzione tende a infinito man mano che la variabile si avvicina a un punto finito X0.

Definizione Generale

Il limite di una funzione f(x) quando x si avvicina a X0 è il valore L che f(x) assume quando x si avvicina sempre più a X0.

Definizione Specifica

La definizione formale di limite è la seguente: per ogni ε > 0 esiste un δ > 0 tale che per ogni x, 0 < |x - X0| < δ, valga |f(x) - L| < ε.
